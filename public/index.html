Toy Routes

Method URL Description
Get /toy/all Get a list of all the Toy
Get /toy/prices?min=minPrices&max=maxPrices Get a list of the toys by min & max .
Get /toy/category/:category-name Get all Toy from the same category
Get /toy/search?s=keyword Search products by name ,color or info
Get /toy/myshop/:{userId} Get all Toy of your User only. Must to be connected!
